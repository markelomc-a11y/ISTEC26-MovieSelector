🎬 Cinema Gótico — Movie Selector & Edição Maligna

Acervo sombrio de filmes com editor visual de posters interativo.


Descrição
Cinema Gótico é uma aplicação web com duas vertentes: um selector de filmes filtráveis por ano, e um editor visual de posters com efeitos de distorção em tempo real. O utilizador pode navegar pelo acervo sombrio, selecionar qualquer filme, carregar o poster para edição e aplicar efeitos visuais arrastando o rato ou manipulando sliders.
O resultado pode ser guardado como imagem.

Funcionalidades
Acervo de Filmes

Listagem de filmes com estética gótica
Filtro por ano ("Ano Maldito")
Acesso rápido ao editor a partir de qualquer filme

Editor Visual (Tormento Visual)

Carregar poster do filme para o canvas
Arrastar o rato sobre a imagem para aplicar distorções em tempo real
Sliders de controlo para intensidade dos efeitos
Guardar a imagem editada localmente (💾 Salvar Imagem Maldita)


Controlos do Editor
Mouse (arrastar) → distorce a imagem na área do cursor
Sliders          → ajustam a intensidade e tipo de tormento visual
Botão Guardar    → exporta a imagem resultante para ficheiro

Tecnologias

HTML5 Canvas
CSS3
JavaScript (vanilla)
